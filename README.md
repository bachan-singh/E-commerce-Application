## Installation & Setup
1. Clone the repository:
   ```sh
   git clone https://github.com/bachan-singh//E-commerce-Application.git
   ```
2. Navigate to the project directory:
   ```sh
   cd client
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm run dev
   ```
